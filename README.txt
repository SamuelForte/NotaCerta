Nota Certa — Protótipo (estático)
================================

Como usar
1. Baixe o arquivo `nota-certa-static.zip` e extraia.
2. Abra o arquivo `index.html` no Chrome (duplo clique funciona).
   - O app usa `localStorage` para guardar suas notas no navegador.
   - Para apagar todos os dados use o botão "Limpar" no painel.

Observações
- Este é um protótipo front-end estático (sem backend). Arquivos são convertidos para data URLs e armazenados localmente.
- Para uso em produção, recomendo configurar back-end para armazenamento seguro dos arquivos e autenticação.

Arquivos incluídos:
- index.html  (aplicação pronta para abrir no Chrome)
